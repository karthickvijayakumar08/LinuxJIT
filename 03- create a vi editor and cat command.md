# ✅ **VI Editor + CAT Command — Questions & Answers**

## **Q1. Create a file using `vi` and display its content using `cat`.**


Create a file named `demo.txt` using the `vi` editor with the text:

```
This is a demo file
Created using vi editor
```

Save the file and then display the contents using the `cat` command.

### **Answer**

**Step 1 – Open file in vi**

```bash
vi demo.txt
```

**Step 2 – Insert text inside vi**
Press:

```
i
```

Type:

```
This is a demo file
Created using vi editor
```

Save and exit:

```
Esc :wq
```

**Step 3 – View file using cat**

```bash
cat demo.txt
```

**Expected Output**

```
This is a demo file
Created using vi editor
```

## **Q4. Create a multi-line file using `vi` and show line numbers using `cat -n`.**

### **Question**

Create a file named `lines.txt` using vi with this content:

```
line1
line2
line3
```

Then display the contents with line numbers.

### **Answer**

```bash
vi lines.txt
```

Inside vi → Insert mode (`i`):

```
line1
line2
line3
```

Save and exit:

```
Esc :wq
```

**Display with line numbers**

```bash
cat -n lines.txt
```

**Expected Output**

```
     1  line1
     2  line2
     3  line3
```

## **Q5. Overwrite a file created by `vi` using `cat > filename`.**

### **Question**

Overwrite the file `demo.txt` so that it contains only:

```
New content replaced
```

Use `cat >` and then display the file content.

### **Answer**

```bash
cat > demo.txt
```

Type:

```
New content replaced
```

Press:

```
Ctrl + D
```

**Now display**

```bash
cat demo.txt
```

**Expected Output**

```
New content replaced
```
